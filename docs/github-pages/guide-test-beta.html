Guide de test utilisateur — Actograph V3 (Bêta)

Tout d'abord, un grand merci pour votre participation aux tests d'Actograph V3 !

Le développement est bien avancé, mais à ce stade Actograph a besoin de vous. Quel que soit votre niveau d'expérience dans l'utilisation d'un outil de relevé et en création de chroniques d'activité, votre retour en tant qu'utilisateur·trice nous sera précieux. La plus grande considération sera accordée à chacune de vos contributions.

📅 De préférence avant le 30 juillet 2026

💡 Si un bouton ou un lien ne réagit pas : téléchargez le fichier puis ouvrez-le dans votre navigateur (double-clic) — tout fonctionnera normalement.

Comment utiliser ce guide ?
  • Choisissez un ou plusieurs chapitres, selon le temps dont vous disposez.
  • Après le Préambule, puis le Chapitre 0. Page d'accueil, vous trouverez un chapitre parcourant les étapes de réalisation d'une chronique : 0 bis. Parcours à partir de la chronique « exemple ». Les autres concernent une étape : I. Protocole, II. Observation in situ, II bis. Vidéo, III. Graphe, IV. Statistiques. Enfin le chapitre M est destiné à tester la version Mobile Android (Apple à venir).
  • Il n'est pas obligatoire de répondre dans l'ordre des chapitres, ni des rubriques (seul le préambule est indispensable). Chaque ligne des tableaux est un test souvent indépendant, avec sa propre durée estimée.
  • Les tests sur fond jaune correspondent à des fonctions « bonus », pas encore garanties dans cette version bêta : testez-les si vous en avez l'occasion.
  • En fin de chaque chapitre, une rubrique permet de faire des retours libres : « Autres tests ».
  • Pour joindre une capture d'écran (ça nous aide beaucoup) : cliquez dans la case dédiée puis collez (Ctrl+V), ou utilisez le lien « choisir un fichier ».
  • Vous pouvez nous faire plusieurs retours échelonnés : vous pouvez renvoyer ce document plusieurs fois à Actograph‑info, avec différents tests ou différentes réponses.

Date limite de retour : 30 juillet 2026, à adresser à actograph-info@symalgo-tech.com. Après cette date, les retours (ou compléments) seront encore considérés, mais ils feront l'objet d'une vague ultérieure de traitement.

Préambule — Quel utilisateur ou utilisatrice êtes-vous ?

Indispensable pour situer votre point de vue. Pour la suite du guide, c'est libre : répondez aux chapitres de votre choix.

Commentaire sur le sujet de votre choix

Merci !

Une fois vos réponses saisies : cliquez sur « Enregistrer » pour générer un PDF de ce guide rempli (la fenêtre d'impression du navigateur s'ouvre — choisissez la destination « Enregistrer au format PDF »). Cliquez ensuite sur « Envoyer par email » : votre messagerie s'ouvrira avec le destinataire et un résumé pré-remplis — pensez à joindre le PDF que vous venez d'enregistrer avant d'envoyer.
